# Ordered list

Ol is a pure styled component render. Renders a standard `<ol>` with themed styling

-------

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import { Ol } from '@salo/core-ui/Typography/ol';
```

Implement as follows

```javascript
<Ol>
  // My content here
</Ol>
```
