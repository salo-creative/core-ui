# Heading three

Heading one is a pure styled component render. Renders a standard `<h3>` with themed styling

---

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import H3 from '@salo/core-ui/Typography/h3';
```

Implement as follows

```javascript
<H3>This is a title</H3>
```
