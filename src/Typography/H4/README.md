# Heading four

Heading one is a pure styled component render. Renders a standard `<h4>` with themed styling

---

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import H4 from '@salo/core-ui/Typography/h4';
```

Implement as follows

```javascript
<H4>This is a title</H4>
```
