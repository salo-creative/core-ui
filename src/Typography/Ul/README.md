# Unordered list

Ul is a pure styled component render. Renders a standard `<ul>` with themed styling

-------

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import { Ul } from '@salo/core-ui/Typography/ul';
```

Implement as follows

```javascript
<Ul>
  // My content here
</Ul>
```
