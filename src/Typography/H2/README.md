# Heading two

Heading one is a pure styled component render. Renders a standard `<h2>` with themed styling

---

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import H2 from '@salo/core-ui/Typography/h2';
```

Implement as follows

```javascript
<H2>This is a title</H2>
```
