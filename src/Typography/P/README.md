# Paragraph

Paragraph is a pure styled component render. Renders a standard `<p>` with themed styling

-------

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import { P } from '@salo/core-ui/Typography/p';
```

Implement as follows

```javascript
<P>This is a title</P>
```
