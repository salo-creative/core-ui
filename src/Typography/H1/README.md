# Heading one

Heading one is a pure styled component render. Renders a standard `<h1>` with themed styling

---

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import H1 from '@salo/core-ui/Typography/h1';
```

Implement as follows

```javascript
<H1>This is a title</H1>
```
