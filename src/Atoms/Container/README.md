# Container

Container is a pure styled component render. Renders a standard `<div>` with themed styling

-------

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import Container from '@salo/core-ui/Atoms/Container';
```

Implement as follows

```javascript
<Container>
  // My content
</Container>
```
