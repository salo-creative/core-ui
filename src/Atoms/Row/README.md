# Row

Row is a pure styled component render. Renders a standard `<div>` with themed styling

-------

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import Row from '@salo/core-ui/Atoms/Row';
```

Implement as follows

```javascript
<Row>
  // My content
</Row>
```
