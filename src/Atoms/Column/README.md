# Column

Column is a pure styled component render. Renders a standard `<div>` with themed styling

-------

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import Column from '@salo/core-ui/Atoms/Column';
```

Implement as follows

```javascript
<Column>
  // My content
</Column>
```
