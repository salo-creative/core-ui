# Card

Card is a pure styled component render. Renders a standard `<div>` with themed styling for a card

-------

## Usage

Install

```javascript
yarn add @salo/core-ui
```

Include

```javascript
import Card from '@salo/core-ui/Atoms/Card';
```

Implement as follows

```javascript
<Card>
  // My content
</Card>
```
